# BlueSkiesWeather

BlueSkiesWeather is a modern and intuitive weather application that provides real-time forecasts, hourly updates, and extended outlooks for any location worldwide. Designed with simplicity and clarity in mind, it helps you stay prepared for the day ahead—whether it’s sunshine, rain, or stormy skies.

# Tech Stack

- [ReactJs](https://react.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Sass](https://sass-lang.com/)
- [Zustand](https://zustand-demo.pmnd.rs/)
- [Vite](https://vite.dev/)
- [React Router](https://reactrouter.com/)
- [Date Fns](https://date-fns.org/)
